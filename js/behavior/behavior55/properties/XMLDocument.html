XMLDocument版本：DHTML Object Model　只读
语法：
Scripting oXMLObject = Object.XMLDocument
取值：
oXMLObject : 对象(Object)。对对象提供的 XML DOM 的引用。
说明：
获取对由对象所提供的 XML 文档对象模型(DOM)的引用。
当数据持续行为被应用到一个要素时，在它所提供的XML DOM中会自动建立一个根节点。这个节点可以通过 documentElement 特性访问。你可以使用 setAttribute 添加属性到根节点。还可以使用 appendChild 添加子节点到根节点。
当 onload 和 onsave 事件触发时，此特性仅仅对 saveFavorite 和 saveHistory 行为可用。
