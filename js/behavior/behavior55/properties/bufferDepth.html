bufferDepth版本：DHTML Object Model　只读
语法：
Scripting iBitsPerPixel = Object.bufferDepth
取值：
iBitsPerPixel : 整数值(Integer)。
0 : 默认值。无外部缓冲发生。 colorDepth 特性设置为屏幕深度。
-1 : 缓冲区深度依据屏幕深度调整。 colorDepth 特性设置为屏幕深度。
1, 4, 8, 15, 16, 24, 32 : 离屏缓冲依据指定的位/象素的值发生。 colorDepth 特性也被设为相同的值。 15 等于16位/象素。因为在 5-5-5 的 红-绿-蓝 布局里只使用了 15 位。
说明：
检索离屏位图缓冲区的色彩每个象素所使用的位数。
不被支持的值会导致此特性值返回 -1 。
当此特性值为 -1 并且用户改变了影响显示深度的系统设置时，缓冲区深度会自动更新到新的深度。但是当此特性值为有效值时，前述的情况不会发生。
可参阅 screen 对象的 bufferDepth 特性。
