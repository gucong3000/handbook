startDownload版本：DHTML Object Model　返回值：无
语法：
Scripting Object .startDownload ( sUrl , fpCallback )
参数：
sUrl : 必选项。字符串(String)。指定下载文件的地址。
fpCallback : 必选项。指定下载完成后要执行的代码的函数的指针。
返回值：
无
说明：
下载指定的文件。
fpCallback 参数指向的回调函数需要指定一个唯一的参数。当文件下载成功后，文件内容会作为此参数的值传递给函数。
此方法仅仅对于文本文件而言返回的是文档的全部文本内容，假如被下载的文件是其他的格式，则返回的是此格式而非文件内容。
请注意，当 fpCallback 参数指向的回调函数是 Microsoft® Visual Basic® 脚本时，您需要使用 GetRef 函数获取此回调函数的指针。请参看下面的示例代码。
示例：
<script language="VBScript">
function onDownloadDone(s)
MsgBox s
End Function
</script>

<rdl:download id="oDownload" style="behavior:url(#default#download)" />
<div onclick="oDownload.startDownload('download.htm',GetRef('onDownloadDone')"> 下载 </div>
