addDABehavior版本：DHTML Object Model　返回值：无
语法：
Scripting Object .addDABehavior ( oBehavior , iID )
参数：
oBehavior : 必选项。对象(Element)。指定 DABehavior 要被添加到的对象。
iID : 必选项。整数(Integer)。指定要被添加的 DABehavior 的标识。此标识用于以后移除此 DABehavior 的 removeDABehavior 方法调用。
返回值：
无
说明：
添加不在动画制作模型中的 Microsoft® DirectAnimation® behavior (DABehavior) 到运行列表。
当系统开始动画时，附加的 DABehavior 会和源动画中的行为在同一时间开始运行。
