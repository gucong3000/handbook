isHomePage版本：DHTML Object Model　返回值：有
语法：
Scripting bHomePage = Object .isHomePage ( sURL )
参数：
sURL : 必选项。字符串(String)。 指定和客户端的默认主页比较的路径和(或)文件名。
返回值：
bHomePage : 布尔值(Boolean)。false | true
false : sURL 是客户端指定的默认主页。
true : sURL 不是客户端指定的默认主页。
说明：
探测 sURL 是否客户端指定的默认主页。
假如调用此方法的页面地址和客户端当前的默认主页不在同一个域内则此方法总是返回 false 。例如，如果 http://www.dhtmlet.com 是您的默认主页，而您在 http://www.dhtmlet.com/demo/index.html 页面内调用此方法，则会返回 true 。而您若从其他域名下的页面内调用此方法则会返回 false 。
