ondocumentready事件

当包含该行为的文档其内容解析完陈时，该事件发生。

格式

<PUBLIC:ATTACH
  event = "ondocumentready"
  for = "sObject"
  id = "sID"
  onevent = "sEventHandler"
/>

属性

EVENT

必需。字符串值；指定动态HTML(DHTML)事件名称，或任何在HTC参考中列举出来的适用于HTML组件(HTC)的事件的名称。

FOR

可选。字符串值；用于指定事件源的下列值中的一个。

document document对象的引用。
element 默认值。附加有该行为的元素的引用。
window window对象的引用。
ID

可选。字符串值；在组件内用于识别该PUBLIC:ATTACH元素的唯一值。这个属性类似于DHTML中的ID属性。

ONEVENT

必需。字符串值；指定一段行内脚本或事件直接调用的处理函数。

注意

ondocumentready和oncontentready两个事件的通告均会被附加到元素的行为接收到，但必须在window.onload事件发生之后。

要对一份HTC文件和它的主文档在初始化过程中事件的发生有一个大致了解，请阅读《组件初始化》。
