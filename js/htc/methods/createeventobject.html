createEventObject方法

创建一个event对象，其作用在于将事件的相关信息传递给PUBLIC:EVENT元素的fire方法。

格式

oEvent = element.createEventObject()

返回值

返回一个event对象。

注意

同一event对象无法多次调用其fire方法。

使用createEventObject创建的event对象的所有属性都是可读写的，包括那些在动态HTML(DHTML)中只读的属性。这让你可以在事件发生前设置来自该HTML组件(HTC)文件的event对象的属性。

范例

此例源自于一个计算器行为样例。只要结果改变，该HTC发生一个自定义的onResultChange事件返回到页面，同时将结果作为event对象的一个自定义属性进行传递。

<PUBLIC:EVENT name="onResultChange" id="rcID" />

<SCRIPT language="JScript">
function doCalc()
{
  :
  oEvent = createEventObject();
  oEvent.result = sResult;
  rcID.fire (oEvent);
}
