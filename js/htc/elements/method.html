PUBLIC:METHOD元素

定义一个该HTC的专有方法，并将其暴露给包含此HTC的文档。

格式

<PUBLIC:METHOD
  id = "sID"
  internalname = "sInternalName"
  name = "sName"
/>

属性

ID

可选。字符串值；在组件内用于识别该PUBLIC:METHOD元素的唯一值。这个属性类似于DHTML中的ID属性。

INTERNALNAME

可选。字符串值；指定该方法在组件内访问的名称。如果没有指定值，默认使用NAME属性。

NAME

必需。字符串值；指定该方法在包含文档中访问的名称。默认的，该NAME属性同样用于在组件内访问，除非指定了INTERNALNAME属性。

元素信息

父元素 PUBLIC:COMPONENT
子元素 无
最低可用版本 Internet Explorer 5
最低操作系统版本 Windows 95，Windows NT 4.0

注意

通过指定与该元素已定义的标准方法同名的NAME属性，该行为可以重载该元素的默认行为。

范例

此例使用该PUBLIC:METHOD标签将startFlying方法从HTC暴露给其包含文档。

<PUBLIC:COMPONENT name="toFly">
  <PUBLIC:METHOD name="startFlying" />

  <SCRIPT LANGUAGE="JScript" >
    function startFlying()
    {
      // Insert flying code here
    }
  </SCRIPT>
</PUBLIC:COMPONENT>

此包含文档的范例示范了调用该方法的一种途径。

<HTML>
  <HEAD>
    <STYLE>
      .FLY {behavior:url(fly.htc)}
    </STYLE>
  </HEAD>

  <BODY id="flyingHigh" class="FLY">
    <DIV onclick="flyingHigh.startFlying()">
      <!-- Insert stuff to fly -->
    </DIV>
  </BODY>
</HTML>
