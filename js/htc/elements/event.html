PUBLIC:EVENT元素

定义一个该HTC的专有事件，并将其暴露给包含此HTC的文档。

格式

<PUBLIC:EVENT
  id = "sEventID"
  name = "sName"
/>

属性

ID

可选。字符串值；在组件内用于识别该PUBLIC:EVENT元素的唯一值。这个属性类似于DHTML中的ID属性。

NAME

必需。字符串值；指定暴露给其包含文档的事件名称。

方法

eventID.fire( [oEvent] )

发生该事件到其包含文档。

此方法拥有下面的参数：

oEvent 指定event对象所包含的相关信息的对象。

元素信息

父元素 PUBLIC:COMPONENT
子元素 无
最低可用版本 Internet Explorer 5
最低操作系统版本 Windows 95，Windows NT 4.0

注意

通过指定与该元素已定义的标准事件同名的NAME属性，该行为可以重载该元素的默认行为。

一个行为中定义的事件不会起泡；它们只发生在附加有该行为的元素上。

范例

此例源自于一个计算器行为样例。只要结果改变，该HTC发生一个自定义的onResultChange事件返回到页面，同时将结果作为event对象的一个自定义属性进行传递。

<PUBLIC:EVENT name="onResultChange" id="rcID" />

<SCRIPT language="JScript">
function doCalc()
{
  :
  oEvent = createEventObject();
  oEvent.result = sResult;
  rcID.fire (oEvent);
}
