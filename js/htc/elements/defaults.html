PUBLIC:ATTACH元素

为一份HTC设置默认的属性。

格式

<PUBLIC:DEFAULTS
  canhavehtml = "bCanHaveHTML"
  contenteditable = "bEditable"
  style = "sStyle"
  tabstop = "bIsTabStop"
  viewinheritstyle = "bInheritsStyle"
  viewlinkcontent = "bViewLinkContent"
  viewmastertab = "bViewMasterTab"
/>

属性

canHaveHTML

可选。布尔值；指定为下列值中的一个：

false 该HTC文件定义的标签中的内容不可以包含HTML标记。
true 该HTC文件定义的标签中的内容可以包含HTML标记。
contentEditable

可选。字符串值；指定为下列值中的一个：

inherit 默认值。对象从其父对象处继承内容是否允许被使用者编辑的能力。
false 该HTC文件定义的标签中的内容是不可编辑的。
true 该HTC文件定义的标签中的内容是可编辑的。
style

可选。字符串值；指定该HTC文件定义的标签的样式。

tabStop

可选。布尔值；指定为下列值中的一个：

false 默认值。该HTC文件定义的标签不可以通过TAB移焦激活。
true 该HTC文件定义的标签可以通过TAB移焦激活。
viewInheritStyle

可选。布尔值；指定为下列值中的一个：

false Viewlink不从主HTML文件中继承样式。
true 默认值。Viewlink从主HTML文件中继承样式。
viewLinkContent

可选。布尔值；指定为下列值中的一个：

false 默认值。该HTC文件的文档中的标记不作Viewlink使用。
true 该HTC文件的文档中的标记作Viewlink使用。
viewMasterTab

可选。布尔值；指定为下列值中的一个：

false Viewlink的主元素并不包含在主文档的TAB移焦序列中。
true 默认值。Viewlink的主元素包含在主文档的TAB移焦序列中。

元素信息

父元素 PUBLIC:COMPONENT
子元素 无
最低可用版本 Internet Explorer 5
最低操作系统版本 Windows 95，Windows NT 4.0

注意

要获得更多的关于在一份HTC文件中使用viewLinkContent属性的信息，阅读文章《Viewlink简介》的《创建Viewlink》部分。
