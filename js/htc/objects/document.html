注意

从Internet Explorer 3.0开始，此对象在脚本中可用。

使用该document对象来检查、修改或添加内容到一份HTML文档中，并处理此文档中的事件。在网页中，document对象可以通过window对象的document属性来引用，也可以直接引用。

document对象在主文档中任意时刻都是可用的，但对使用轻量的动态HTML(DHTML)行为的HTML组件(HTC)是不存在的。这是因为轻量的行为只有不在HTC文件中使用一个document对象时才能定义。因此，轻量的行为比一般行为运行更快，也更有效率。然而，轻量的DHTML行为也可以通过和一般DHTML行为相同的方式来访问其主文档的document对象。

当包含了行为的主文档中该document对象可用时，ondocumentready事件向DHTML行为进行通报。一旦发生ondocumentready事件，行为就可以开始对主document对象进行操作。

行为中的脚本可以引用HTC文件或其主文档中的document对象。在HTC文件中进行脚本编码时，使用element.element来访问其主文档的document对象。

范例

此例使用document对象来检查文档标题并在其非空时显示在一个消息框中。

if (document.title!="")
  alert("The title is " + document.title)

此例展示了一个事件处理函数，它将鼠标相对于文档的左上角的位置显示在浏览器的状态条中。

<HTML>
<HEAD><TITLE>Report mouse moves</TITLE>
<SCRIPT language="JScript">
function reportMove()
{
  window.status = "X=" + window.event.x + " Y=" + window.event.y;
}
</SCRIPT>
<BODY onmousemove="reportMove()">
<H1>Welcome!</H1>
</BODY>
</HTML>
