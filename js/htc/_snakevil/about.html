我就是我

 
其人

Snakevil，男，湖北仙桃人氏，出生于1984年8月6日。缺乏悟性，仅仅是纯粹的XML，其学习时间便长达三个月之久。行事慵懒，喜欢呆在床上，尤其是在写程序或玩游戏的时候。性格乖僻，时常放着现有的小工具程序不用，自己偏偏要去写一个，诸如GUID Generator和JScript Obfuscater便是典型的例子。自我矛盾，一个搞网络开发有五年经验的人居然没有自己的个人网站甚至是BLOG。

不过幸而谈吐随和，蒙众多朋友关爱不为所弃，并送诨号“大蛇”。

 
其言

正所谓“年少轻狂”，尚在大学之时，时常自以为老子天下第一。后经过连番挫折，终于明白“天外有天，人外有人”之所指。遂在“路漫漫其修远兮，吾将上下而求索”自励之余，再添一句“态度决定一切”以自警。

 
其生

读高中时得以接触网络，进而一发不可收拾，看着花花绿绿会自己动的网页，总有一股欲念让自己也想学会怎么去制作它。

进了大学，日子就变得海阔天空起来。对着自己的电脑，学习的念头也愈发变得强烈，于是乎我便住进了无忧脚本，日夜向出没于其间的诸多高手请教着、学习着。

对HTML、CSS和JScript这些客户端技术有了相对深入的了解之后，突然感觉到有些无趣，这些对我而言本来就是远远不足。于是乎在论坛了解了更多的ASP知识后，我开始了这门技术的学习。而在数据仓库方面，因为对原文文档断章取义的理解，我曾经在一段时间内固执的认为SQL技术已经要被XML淘汰了。看起来很顺理成章的，我学习了XML以及与之联系紧密地XSLT、XPath、XML Schema等等。

此段时间内因为有着充裕的自由发挥空间，我先后完成了很多演示作品。诸如被我命名为ERiW.1.0的MSDN Catalog Tree的完美模拟、名为ERiW.0.2的放置在zephyr.8u8.com空间的全XML + XSLT + CSS样例站点。而也正是因为反复的加深理解和尝试后的归纳整理，我熟练地掌握了这些技术，并有机会接触到MSDN中更多的技术，如HTC和HTA，还有SVG的前身VML。

随着ASP.net技术的普及，我也变得越来越彷徨——我是否也应该学着其他人与时俱进呢？！偶然的机会，我接触到PHP这种类似于ASP的技术。虽然最开始抱着玩一玩打发时间的心态去学习，可随着层次的深入，我发现我越来越热爱它，于是乎又一次顺理成章的，Apache服务器、MySQL数据这些相关技术我也进行了学习，并住进了另外一个新家喜悦国际村。传说中的自成体系的的四大技术L.A.M.P.，除了Linux因为条件限制没机会尝试，其它三者都被我磕上了。

再后来通过PHP的工作实践，我对大型项目的软件工程开发、面向对象的设计模式、公共网站的高压解决方案等等以前自娱自乐时所无法涉及的技术领域都有了自己不同层次的理解。

以后呢？不知道，程序开发应该会作为我的一种乐趣一直保留下去吧。

Deflation 擅长 Apache和PHP中都可以实现此目的。因为南北互通的问题，让北方用户访问南方服务器上150K甚至更大的首页面总是一件无法让人愉快地事情。因此我实践了我的想法，很成功。
Cache Control 擅长 在大型公共网络系统的实际运用中，从数据的采集到最终用户的浏览，其实可以加入多达6层的缓存处理。别人进行了详细的整理，而我则幸运的看到了。在公司网站的实际运行中，我也为提高性能加入了尽可能恰当的缓存控制。
SQL Optimization 擅长 在一个涉及数据库多达40张分表的程序中，我成功地将原有的50多次查询，利用采集镜像表和MySQL中的UNION语法，缩减到8次查询。实际运行的结果是效率提高到原来的700%倍。
