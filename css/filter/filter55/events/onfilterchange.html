onfilterchange版本：DHTML Object Model
语法：
HTML <element  onfilterchange = " Scripts ">
HTML <script  for = object  event = onfilterchange>  Scripts  </script>
Scripting object.onfilterchange = Handle
Scripting function object.onfilterchange () { Scripts }
信息：
事件冒泡(Bubbles) : 否
可被取消(Cancels) : 否
激活条件(To invoke) : 滤镜(Visual Filters and Transitions)状态改变时。
默认动作(Default action) : 发出滤镜状态改变的信号。
说明：
当滤镜(Visual Filters and Transitions)状态改变或完成时激发的事件。
