direction | Direction版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.sFilterName ( direction = iOffset )
Scripting object.filters.item('DXImageTransform.Microsoft.sFilterName').Direction = iOffset
取值：
iOffset : 整数值(Integer)。默认单位为角度。0 | 45 | 90 | 135 | 180 | 225 | 270 | 315
0 : 上。
45 : 右上。
90 : 下。
135 : 右下。
180 : 下。
225 : 默认值。左下。
270 : 左。
315 : 左上。
说明：
设置或检索滤镜效果的运动偏移方向。小于 0 或大于 360 的值会自动转换为 0 - 360 之间的值。例如， -45 会转换为 315 。
