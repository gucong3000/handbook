FilterType | FilterType 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:FilterTypeImageTransform.Microsoft.Matrix ( FilterType = sType )
Scripting object.filters.item('FilterTypeImageTransform.Microsoft.Matrix').FilterType = sType
取值：
sType : 字符串(String)。bilinear | nearest neighbor
bilinear : 默认值。使用双线性( bilinear )算法定义目标像素的色彩内容。
nearest neighbor : 使用领近相似( nearest neighbor )算法定义目标像素的色彩内容。
说明：
设置或检索定义新内容的像素的方法。
要想获得最好的平滑的静态滤镜效果，请使用 bilinear 方法参数。而当用于动画图像时， nearest neighbor 方法参数因其运算速度较快从而能够提供动画每秒更多图像。
