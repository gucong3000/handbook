语法：
CSS filter : progid:FilterTypeImageTransform.Microsoft.AlphaImageLoader ( sizingMethod = sMethod )
Scripting object.filters.item('FilterTypeImageTransform.Microsoft.AlphaImageLoader').sizingMethod = sMethod
取值：
sMethod : 字符串(String)。
crop : 剪切图片以适应对象尺寸。
image : 默认值。增大或减小对象的尺寸边界以适应图片的尺寸。
scale : 缩放图片以适应对象的尺寸边界。
说明：
设置或检索滤镜作用的对象的图片在对象容器边界内的显示方式。
