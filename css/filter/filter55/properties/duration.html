duration | Duration 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.sFilterName ( duration = fDuration )
Scripting object.filters.item('DXImageTransform.Microsoft.sFilterName').Duration = fDuration
取值：
fDuration : 浮点数(Real)。其值为秒.毫秒(0.0000)格式。
说明：
设置或检索转换完成所用的时间。你可以使用 play 方法的 iDuration 参数设置转换回放的持续时间。然而，当你一旦调用了 play 方法，在回放持续过程中 Duration 特性就变为只读特性。
