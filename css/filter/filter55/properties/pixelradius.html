pixelRadius | PixelRadius 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.Blur ( pixelRadius = flRadius )
Scripting object.filters.item('DXImageTransform.Microsoft.Blur').PixelRadius = flRadius
取值：
flRadius : 浮点数(Float)。默认单位为像素(px)。
取值范围为 1.0 - 100.0 。默认值为 2.0 。当此参数值小于 1.0 时滤镜失去作用。当此参数值大于 100.0 时，此参数使用默认值 2.0 。
说明：
设置或检索模糊效果的作用深度。
