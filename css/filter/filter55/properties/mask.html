mask | Mask版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.BasicImage ( mask = bMask )
Scripting object.filters.item('DXImageTransform.Microsoft.BasicImage').Mask = bMask
取值：
bMask : 布尔值(Boolean)。0 | 1
1 : 添加遮罩。
0 : 默认值。显示对象的原始色彩。
说明：
设置或检索是否依据 MaskColor 的值为对象添加遮罩(Mask)。
示例：
oDiv.filters['DXImageTransform.Microsoft.BasicImage'].Mask=1;
oDiv.filters['DXImageTransform.Microsoft.BasicImage'].MaskColor=0xFF9900;
