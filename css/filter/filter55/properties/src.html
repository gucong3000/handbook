语法：
CSS filter : progid:FilterTypeImageTransform.Microsoft.AlphaImageLoader ( src = sURL )
Scripting object.filters.item('FilterTypeImageTransform.Microsoft.AlphaImageLoader').src = sURL
取值：
sURL : 字符串(String)。绝对或相对 url 地址。
说明：
使用绝对或相对 url 地址指定背景图像。假如忽略此参数，滤镜将不会作用。
载入的图片会成为对象的显示区域。你可以通过设置 AlphaImageLoader 滤镜的 sizingMethod 属性值缩放或剪切图片。
