overlap | Overlap 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.Fade ( overlap = fOverlap )
Scripting object.filters.item('DXImageTransform.Microsoft.Fade').Overlap = fOverlap
取值：
fOverlap : 浮点数(Real)。取值范围为 0.0 - 1.0 。默认值为 1.0 。
说明：
设置或检索在转换进程中源内容和目标内容都被显示的比例。
假如此属性值设为 0.4 ， duration 属性值设为 10 ，则转换会发生下属行为：
  1. 前三秒，源对象开始隐出。
  2. 随后的四秒，源对象完全隐出，新内容同时隐入。
  3. 最后三秒，新内容完全隐入。
