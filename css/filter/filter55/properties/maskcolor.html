MaskColor版本：DHTML Object Model　读/写
语法：
Scripting object.filters.item('DXImageTransform.Microsoft.BasicImage').MaskColor = iColor
取值：
iColor : 整数值(Integer)。32位 ARGB 格式色彩值。
其格式为 0xAARRGGBB 。 AA 、 RR 、 GG 、 BB 为十六进制正整数。取值范围为 00 - FF 。 RR 指定红色值， GG 指定绿色值， BB 指定蓝色值。 AA 指定透明度。 00 是完全透明。 FF 是完全不透明。
取值范围为 0x00000000 - 0xFFFFFFFF 。默认值为 0x00000000 。不透明黑色。
说明：
设置或检索遮罩( Mask )颜色。
示例：
oDiv.filters['DXImageTransform.Microsoft.BasicImage'].Mask=1;
oDiv.filters['DXImageTransform.Microsoft.BasicImage'].MaskColor=0xFF9900;
