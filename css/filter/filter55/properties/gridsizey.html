gridSizeY | GridSizeY 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DYImageTransform.Microsoft.sFilterName ( gridSizeY = iColumns )
Scripting object.filters.item('DYImageTransform.Microsoft.sFilterName').GridSizeY = iColumns
取值：
iColumns : 整数值(Integer)。取值范围为 1 - 100 。默认值为 16 。
说明：
设置或检索滤镜效果中纵向盘旋多少次。
