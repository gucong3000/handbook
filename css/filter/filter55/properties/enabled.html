enabled | Enabled 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.sFilterName ( enabled = bEnabled )
Scripting object.filters.item('DXImageTransform.Microsoft.sFilterName').Enabled = bEnabled
取值：
bEnabled : 布尔值(Boolean)。true | false
true : 默认值。滤镜激活。
false : 滤镜被禁止。
说明：
设置或检索滤镜是否激活。改变此属性值会激发 onfilterchange 事件。
