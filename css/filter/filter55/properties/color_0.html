Color | Color版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.Shadow ( Color = sColor )
Scripting object.filters.item('DXImageTransform.Microsoft.Shadow').Color = sColor
取值：
sColor : 字符串(String)。其格式为 #RRGGBB 。 RR 、 GG 、 BB 为十六进制正整数。取值范围为 00 - FF 。 RR 指定红色值， GG 指定绿色值， BB 指定蓝色值。
说明：
设置或检索 Shadow 滤镜作用的颜色值。
