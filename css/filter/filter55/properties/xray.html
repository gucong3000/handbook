XRay | XRay版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.BasicImage ( XRay = bXRay )
Scripting object.filters.item('DXImageTransform.Microsoft.BasicImage').XRay = bXRay
取值：
bXRay : 布尔值(Boolean)。0 | 1
1 : 以X光效果显示对象内容。
0 : 默认值。正常显示对象内容。
说明：
设置或检索是否通过红、绿均衡补偿的方法以X光效果显示对象内容。
