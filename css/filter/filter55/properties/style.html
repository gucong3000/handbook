style | Style版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.Alpha ( style = iStyle )
Scripting object.filters.item('DXImageTransform.Microsoft.Alpha').Style = iStyle
取值：
iStyle : 整数值(Integer)。0 | 1 | 2 | 3
0 : 默认值。整体透明度。将 Opacity 值均匀的作用于对象。
1 : 线性渐变透明度。从由 StartX 和 StartY 决定的点，由 Opacity 决定的透明度开始，到由 FinishX 和 FinishY 决定的点，由 FinishOpacity决定的透明度结束。
2 : 圆形放射渐变透明度。圆形放射区域的圆心为对象的中心，圆周到与对象的边相切为止。透明渐变由圆心开始，到圆周结束。开始透明度由 Opacity 决定，结束透明度由 FinishOpacity 决定。
3 : 矩形放射渐变透明度。由对象的边开始，到对象的中心结束。开始透明度由 Opacity 决定，结束透明度由 FinishOpacity 决定。
说明：
设置或检索透明渐变的样式。
