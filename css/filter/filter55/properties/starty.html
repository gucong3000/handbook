startY | StartY 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.Alpha ( startY = iPercent )
Scripting object.filters.item('DXImageTransform.Microsoft.Alpha').StartY = iPercent
取值：
iPercent : 整数值(Integer)。其数值作为对象高度的百分比值处理。默认值为 0 。
说明：
设置或检索 Alpha 滤镜透明渐变开始点的垂直坐标。仅当 style 属性值设置为 1 ，即渐变样式为线性(linear)渐变时，此属性才会发生作用。
