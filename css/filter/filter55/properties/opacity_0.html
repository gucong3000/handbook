opacity | Opacity版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.BasicImage ( opacity = fOpacity )
Scripting object.filters.item('DXImageTransform.Microsoft.BasicImage').Opacity = fOpacity
取值：
fOpacity : 浮点数(Float)。取值范围为 0.0 - 1.0 。默认值为 1.0 。不透明。 0.0 为完全透明。
说明：
设置对象内容的透明度。
