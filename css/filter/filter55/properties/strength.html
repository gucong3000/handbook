strength | Strength版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.sFilterName ( strength = iDistance )
Scripting object.filters.item('DXImageTransform.Microsoft.sFilterName').Strength = iDistance
取值：
iDistance : 整数值(Integer)。单位为像素( px )。 此属性无默认值。以下为例外:
MotionBlur : 5
Glow : 5
Wave : 1
说明：
设置或检索以对象为基准的在运动方向上的向外扩散距离。
此属性作用于下列滤镜有不同的取值范围:
MotionBlur : >=0
Glow : 1 - 255
Wave : >=1
