length版本：DOM1　读/写
语法：
Scripting object . length = iLength
取值：
iLength : 整数值(Integer)。无默认值。
说明：
设置或检索集合内的对象数目。
此属性是可读写的。对于下列对象来说，是只读的：
COMMENT pages

window.length 将返回 window 内包含的所有框架(frame)数目。
comment.length 将返回对象内的字符数目。
form.length 返回的计数不包括 input type=image 元素的数目。要访问 form 内的所有对象，使用 children 集合。
虽然此属性值对于大多数对象而言是只读的，但是对于下述对象是可读写的：
areas options select
