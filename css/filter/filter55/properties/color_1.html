color | Color版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.sFilterName ( color = sColor )
Scripting object.filters.item('DXImageTransform.Microsoft.sFilterName').Color = sColor
取值：
sColor : 字符串(String)。32位 ARGB 格式色彩值。
其格式为 #AARRGGBB 。 AA 、 RR 、 GG 、 BB 为十六进制正整数。取值范围为 00 - FF 。 RR 指定红色值， GG 指定绿色值， BB 指定蓝色值， AA 指定透明度。 00 是完全透明。 FF 是完全不透明。
此属性无默认值。以下为例外:
Chroma : #FF0000FF
DropShadow : #FF404040
Glow : #FF0000
MaskFilter : #FF000000
说明：
设置或检索滤镜作用的颜色值。
对于 Glow 滤镜而言，此属性的取值被限制为 #RRGGBB 格式。因为辉光发热效果总是不透明的。
此属性作用于下列滤镜有不同的作用:
Chroma : 指定被透视的颜色。
DropShadow : 指定阴影效果的颜色。
Glow : 指定辉光发热效果的颜色。
MaskFilter : 指定显示为透明的颜色。
