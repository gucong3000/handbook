finishOpacity | FinishOpacity 版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.Alpha ( finishOpacity = iOpacity )
Scripting object.filters.item('DXImageTransform.Microsoft.Alpha').FinishOpacity = iOpacity
取值：
iOpacity : 整数值(Integer)。取值范围为 0 - 100 。默认值为 0 ，即完全透明。 100 为完全不透明。
说明：
设置或检索透明渐变的结束透明度。
