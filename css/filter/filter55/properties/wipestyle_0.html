wipeStyle | WipeStyle版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.RadialWipe ( wipeStyle = sStyle )
Scripting object.filters.item('DXImageTransform.Microsoft.RadialWipe').WipeStyle = sStyle
取值：
sStyle : 字符串(String)。CLOCK | WEDGE | RADIAL
CLOCK : 默认值。围绕对象的中心，自上方开始，顺时针旋转擦除。
WEDGE : 围绕对象的中心，自上方开始，同时向两边旋转擦除。
RADIAL : 以对象的左上角为圆心旋转擦除。
说明：
设置或检索转换所使用的擦除方式。
