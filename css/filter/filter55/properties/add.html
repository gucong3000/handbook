add | Add版本：DHTML Object Model　读/写
语法：
CSS filter : progid:DXImageTransform.Microsoft.sFilterName ( add = bAddImage )
Scripting object.filters.item('DXImageTransform.Microsoft.sFilterName').Add = bAddImage
取值：
bAddImage : 布尔值(Boolean)。true | false
true : 滤镜作用图像覆盖原始图像。
false : 默认值。只显示滤镜作用图像。
说明：
设置或检索滤镜作用图像是否覆盖原始图像。
