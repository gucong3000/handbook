item版本：DHTML Object Model　返回值：有
语法：
Scripting oItem = object . item ( vIndex )
参数：
vIndex : 必选项。整数值(Integer) | 字符串(String)
如果此参数使用的是整数值，则返回数组中的整数序号等于此参数值的对象。如此参数值等于 0 则返回数组中的第一个对象。
如果此参数值使用的是字符串值，则返回数组中所有 name 或 id 属性等于此参数值的对象。假如符合条件的对象数目多于一个，则返回的将是一个数组。
返回值：
oItem : 必选项。对象(Object)。假如成功返回获取到的对象或数组。否则返回 null 。
说明：
从 filters 集合或其他集合中返回符合条件的对象。
示例：
oDiv.filters.item('DXImageTransform.Microsoft.Light').clear();
oDiv.filters['DXImageTransform.Microsoft.Light'].clear();
oDiv.filters.item(0).clear();
oDiv.filters.item[0].moveLight(0,180,60,20,false);
