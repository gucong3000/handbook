namedItem版本：DOM1　返回值：有
语法：
Scripting oItem = object . namedItem ( sName )
参数：
sName : 必选项。字符串(String)。
返回值：
oItem : 对象(Object)或数组(Array)。如果不成功返回 null 。
说明：
返回集合中 name 属性值或 id 属性值等于 sName 的对象或一组对象的集合。
此方法先在集合中搜寻 id 属性值等于 sName 的对象，如果没有找到，再搜寻 name 属性值等于 sName 的对象。但只获取本身拥有 name 属性和 id 属性的对象。使用自定义( expando )的 name 属性的对象不会被获取。
