TextNode 对象

将文本字符串代表为文档层次中的结点。

成员表

下面的表格列出了 TextNode 对象引出的成员。请单击左侧的标签来选择你想要查看的成员类型。

属性

属性描述
data 设置或获取 TextNode 对象的值。
length 获取 TextNode 对象中的字符数目。
nextSibling 获取对此对象的下一个兄弟对象的引用。
nodeName 获取特定结点类型的名称。
nodeType 获取所需结点的类型。
nodeValue 设置或获取结点的值。
parentNode 获取文档层次中的父对象。
previousSibling 获取对此对象的上一个兄弟对象的引用。

方法

方法描述
appendData 向对象末尾添加新的字符串。
deleteData 从对象中删除指定范围的字符。
insertData 在对象的指定偏移量处插入新字符串。
replaceData 将对象中指定范围的字符替换为新字符串。
splitText 在指定索引处分割文本结点。
substringData 从对象中析取一定范围的字符。

注释

Use the createTextNode method to create a TextNode object. Once you create the TextNode, you can add to it using the appendChild, replaceNode, or insertBefore methods.

此对象在 Microsoft® Internet Explorer 5 的脚本中可用。

示例

下面的例子使用 TextNode 对象更改 li 对象的文本。

<SCRIPT>
function fnChangeText(){
var oTextNode = document.createTextNode("新建的列表项目 1");
var oReplaceNode = oItem1.firstChild.replaceNode(oTextNode);
}
</SCRIPT>
<UL onclick = "fnChangeText()">
<LI ID = oItem1>列表项目 1
</UL>

标准信息

此对象定义在 HTML 3.2  中。
