注释

HEAD 元素提供了不影响文档渲染效果但对于浏览器可能非常有用的信息。此元素内可出现下列标签：

  • BASE
  • BASEFONT
  • BGSOUND
  • LINK
  • META
  • nextID
  • SCRIPT
  • STYLE
  • TITLE

此元素在 Microsoft® Internet Explorer 3.0 的 HTML 中可用，在 Internet Explorer 4.0 的脚本中可用。

此元素不会被渲染。

此元素需要关闭标签。

示例

下面的例子使用了 HEAD 元素创建了文档标题。

<HEAD>
<TITLE>一个简单文档</TITLE>
</HEAD>
