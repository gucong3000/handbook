rule 对象

代表了层叠样式表(CSS)中由选择符和一个或多个声明组成的的样式。

成员表

下面的表格列出了 rule 对象引出的成员。请单击左侧的标签来选择你想要查看的成员类型。

属性

属性描述
readOnly 获取规则或样式表是定义在页面上还是导入的。
selectorText 获取标识相应的样式表规则应用到的元素的字符串。

对象

对象描述
style 代表了给定元素所有可能的内嵌样式的当前设置。

注释

rule 对象定义了应用到一组 HTML 元素上的 CSS 属性。例如，一条包括选择器 H1 和声明 font-family:Arial 的规则定义了全部的 H1 元素都要以 Arial 字体渲染。

此对象在 Microsoft® Internet Explorer 5.0 的脚本中可用。

示例

下面的例子使用了包括选择器 H1 的 rule 对象定义了一条规则将文档中的全部 H1 标题都变成红色。

<STYLE>
H1 { color: red }
</STYLE>

标准信息

没有应用于此对象的公共标准。
