HTML 注释对象

避免任何内含文本或 HTML 源代码被处理并在浏览器窗口中显示。

成员表

下面的表格列出了 HTML 注释 对象引出的成员。请单击左侧的标签来选择你想要查看的成员类型。

属性

属性描述
innerHTML 设置或获取位于对象起始和结束标签内的 HTML。
outerHTML 设置或获取对象及其内容的 HTML 形式。
tagName 获取对象的标签名称。
text 设置或获取对象文本的字符串形式。

事件

事件描述
onlayoutcomplete 当打印或打印预览版面处理完成用来自于源文档的内容填充当前 LayoutRect 对象时触发。

注释

注释可以包含其它 HTML 元素，但不能嵌套。

需要开始和结束标签。

示例

下面是 HTML 注释的例子。

<!-- 此文本不会出现在浏览器窗口中。 -->

标准信息

此对象定义在 HTML 3.2  中。
