STYLE 元素 | style 对象

指定页面的样式表。

成员表

下面的表格列出了 style 对象引出的成员。请单击左侧的标签来选择你想要查看的成员类型。

标签属性/属性

标签属性属性描述
disabled 设置或获取是否已为对象应用样式表。
ID id 获取标识对象的字符串。
innerHTML 设置或获取位于对象起始和结束标签内的 HTML。

集合

集合描述
behaviorUrns 返回标识附加到该元素行为的统一资源名称(URN)字符串的集合。

事件

事件描述
onerror 当对象装载过程中发生错误时触发。
onreadystatechange 当对象状态变更时触发。

方法

方法描述
addBehavior 给元素附加一个行为。
dragDrop 初始化拖曳事件。
removeBehavior 分离元素的行为。
removeNode 从文档层次中删除对象。

样式

样式标签属性样式属性描述
MEDIA media 设置或获取媒体类型。
TYPE type 获取样式表编写所用的样式表(CSS)语言。

注释

STYLE 元素只能在 HTML 文档的 HEAD 节内使用。Microsoft® Internet Explorer 4.0 及以后版本允许多个样式块。

此元素在 Microsoft® Internet Explorer 3.0 的 HTML 中可用，在 Internet Explorer 4.0 的脚本中可用。

此元素不会被渲染。

此元素需要关闭标签。

示例

下面的例子在 STYLE 元素中包含了样式声明，然后使用 style 对象更改了其中的设置之一。

<HEAD>
<STYLE>
BODY {  background-color: white; color: black;  }
H1 {  font: 8pt Arial bold;  }
P  {  font: 10pt Arial; text-indent: 0.5in;  }
A  {  text-decoration: none; color: blue;  }
</STYLE>
</HEAD>
<BODY>
<P id=oParagraph>示例段落文本</P>
<SCRIPT>
oParagraph.style.fontSize = 14;
</SCRIPT>
</BODY>

标准信息
