OPTGROUP 元素 | optGroup 对象

允许作者对 select 元素中的选项进行逻辑分组。

成员表

下面的表格列出了 optGroup 对象引出的成员。请单击左侧的标签来选择你想要查看的成员类型。

标签属性/属性

标签属性属性描述
DISABLED disabled 设置或获取可用于实现对象的你自己的 disabled 功能的字符串。
LABEL label 设置或获取选项组的标签。

注释

在 HTML 4.0 中，所有的 OPTGROUP 元素必须在 SELECT 元素中直接指定。组不允许嵌套。

你仅可以向与被创建的 OPTGROUP 元素位于同一窗口的 SELECT 元素添加 OPTGROUP。

此元素在 Microsoft® Internet Explorer 6 的 HTML 中可用。

此元素是块元素。

此元素需要关闭标签。

示例

下面的例子演示了如何使用 OPTGROUP 元素创建下拉列表框中的项目组。

<SELECT>
<OPTGROUP LABEL="碱性金属">
<OPTION>锂 (Li)</OPTION>
<OPTION>纳 (Na)</OPTION>
<OPTION>钾 (K)</OPTION>
</OPTGROUP>
<OPTGROUP LABEL="卤素">
<OPTION>氟 (F)</OPTION>
<OPTION>氯 (Cl)</OPTION>
<OPTION>溴 (Br)</OPTION>
</OPTGROUP>
</SELECT>

标准信息
