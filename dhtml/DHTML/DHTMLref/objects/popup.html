popup 对象

一种特殊的顶层窗口，主要用于出现在应用程序主窗口之外的对话框、消息框和其它临时窗口。

成员表

下面的表格列出了 popup 对象引出的成员。请单击左侧的标签来选择你想要查看的成员类型。

属性

属性描述
document 获取给定弹出窗口中的 HTML 文档。
isOpen 获取表明弹出窗口是否已打开的值。

方法

方法描述
hide 关闭弹出窗口。
show 在屏幕上显示弹出窗口。

注释

此对象在 Microsoft® Internet Explorer 5.5 的脚本中可用。

示例

下面的例子演示了 popup 对象的使用。该代码创建了弹出窗口并显示之。

<SCRIPT LANGUAGE="JScript">
var oPopup = window.createPopup();
var oPopupBody = oPopup.document.body;
oPopupBody.innerHTML = "在这里显示 <B>HTML</B>。";
oPopup.show(100, 100, 200, 50, document.body);
</SCRIPT>
