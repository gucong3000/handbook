LINK 元素 | link 对象

允许当前文档和外部文档之间建立连接。

注释

LINK 元素只能在 HEAD 标签内使用。

此元素在 Microsoft® Internet Explorer 3.0 的 HTML 中可用，在 Internet Explorer 4.0 的脚本中可用。

此元素不会被渲染。

此元素不需要关闭标签。

示例

下面的例子使用了 LINK 元素为页面应用了一个叫做 styles.css 的外部样式表。

<LINK REL=stylesheet HREF="styles.css" type="text/css">

标准信息

此对象定义在 HTML 3.2  和万维网协会(W3C)文档对象模型(DOM) 级别 1  中。
