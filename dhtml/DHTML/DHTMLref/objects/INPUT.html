INPUT 元素 | input 对象

创建各种表单输入控件。

成员表

下面的表格列出了 input 对象引出的成员。请单击左侧的标签来选择你想要查看的成员类型。

标签属性/属性

标签属性属性描述
ACCEPT accept 设置或获取以逗号分隔的内容类型列表。
ALIGN align 设置或获取对象针对其邻接文本如何排列。
ALT alt 设置或获取用于替代图像的文本。
complete 获取对象是否已完全装入。
DYNSRC dynsrc 设置或获取要在窗口中显示的视频剪辑或 VRML 世界。
HSPACE hspace 设置或获取对象的水平边距。
LOOP loop 设置或获取声音或视频剪辑在激活时的循环播放次数。
LOWSRC lowsrc 设置或获取要显示的较低分辨率图像。
start start 设置或获取视频剪辑文件应该开始播放的时间。
USEMAP useMap 设置或获取用于客户端图像映射的 URL，通常带有一个书签扩展(#name)。
VALUE value 获取控件的默认或选中值。
VSPACE vspace 设置或获取对象的垂直边距。

行为

行为描述
mask 为 DHTML 对象指定遮罩编辑行为。

样式

样式标签属性样式属性描述
layout-flow layoutFlow 设置或获取对象内容的方向和绕排方向。
text-underline-position textUnderlinePosition 设置或获取对象的 textDecoration 属性中设置的下划线的位置。
word-wrap wordWrap 设置或获取当内容超过其容器边界时是否断词。
writing-mode writingMode 设置或获取对象内容的方向和绕排。
zoom zoom 设置或获取对象的放大比例。

注释

INPUT 元素的 TYPE 标签属性用于指定下列表单输入控件之一:

button checkbox file hidden image
password radio reset submit text

此元素是一个内嵌元素。

此元素不需要关闭标签。

示例

下面的例子演示了如何使用 INPUT 元素创建不同类型的输入控件。

<FORM ACTION="http://intranet/survey" METHOD=POST>
<P>姓名</P>
<BR><INPUT NAME="CONTROL1" TYPE=TEXT VALUE="你的姓名">
<P>密码</P>
<BR><INPUT TYPE="PASSWORD" NAME="CONTROL2">
<P>颜色</P>
<BR><INPUT TYPE="RADIO" NAME="CONTROL3" VALUE="0" CHECKED>红色
<INPUT TYPE="RADIO" NAME="CONTROL3" VALUE="1">绿色
<INPUT TYPE="RADIO" NAME="CONTROL3" VALUE="2">蓝色
<P>评论</P>
<BR><INPUT TYPE="TEXT" NAME="CONTROL4" SIZE="20,5" MAXLENGTH="250">
<P><INPUT NAME="CONTROL5" TYPE=CHECKBOX CHECKED>发送收条</P>
<P><INPUT TYPE="SUBMIT" VALUE="确定"><INPUT TYPE="RESET" VALUE="重置"></P>
</FORM>

标准信息

此对象定义在 HTML 3.2  和万维网协会(W3C)文档对象模型 (DOM) 级别 1  中。
