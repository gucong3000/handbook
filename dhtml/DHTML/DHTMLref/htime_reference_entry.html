HTML+TIME 2.0 Reference

HTML+TIME, first released in Microsoft?Internet Explorer 5, adds timing and media synchronization support to HTML pages. Using a few Extensible Markup Language (XML)-based elements and attributes, you can add images, video, and sounds to an HTML page, then synchronize them with HTML text elements over a specified amount of time.
