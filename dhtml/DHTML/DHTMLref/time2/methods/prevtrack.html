prevTrack Method

Internet Development Index

Sets the media-playing container to the previous track, or playItem object, in the playList collection and begins playing that track.

Syntax

object.prevTrack()

Return Value

No return value.

Remarks

Using this method on the first track in the playList causes that track to restart.
