userLanguage Property

Internet Development Index

Retrieves the current user language.

Syntax

XMLN/A
Scripting[ sLanguage = ] clientCaps.userLanguage

Possible Values

sLanguageString that receives any of the possible return values listed in Language Codes.

The property is read-only. The property has no default value.
