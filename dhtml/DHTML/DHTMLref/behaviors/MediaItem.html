MediaItem Object

Internet Development Index

Represents a single item in a media player playlist.

Members Table

The following table lists the members exposed by the MediaItem object. Click a tab on the left to choose the type of member you want to view.

Attributes/Properties

PropertyDescription
attributeCount Sets or retrieves the count of the attributes associated with the media object.
duration Sets or retrieves the duration of the current media item.
name Sets or retrieves the name of the media.
sourceURL Sets or retrieves the URL of the media.

Methods

MethodDescription
getAttributeName Returns the name of the attribute whose index has been specified.
getItemInfo Returns the value of the specified attribute for the current media.

Applies To

mediaBar
