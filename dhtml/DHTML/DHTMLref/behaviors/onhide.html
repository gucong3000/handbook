onhide Event

Internet Development Index

Fires when the Media Bar player is hidden.

Syntax

Event propertymediaBar.onhide = handlerJScript only
mediaBar.onhide = GetRef("handler")Visual Basic Scripting Edition (VBScript) 5.0 or later only
Named script <SCRIPT FOR = mediaBar EVENT = onhide> Internet Explorer only

Event Information

BubblesNo
CancelsNo
To invoke
  • Click the Close button in the Media pane.
  • Click the Media button in the Internet Explorer toolbar.
  • Choose Explorer Bar from the View menu and clear the Media menu item.
Default action Initiates any action associated with this script.
