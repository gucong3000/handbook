创建动态弹出元素

Microsoft® Internet Explorer 5.5 中新的弹出特性提供了创建快捷菜单、对话框、弹出提示和词汇表的便捷方式。全部这些内容都可用 DHTML 创建。

用右键单击主文档的任何位置，都将显示一个使用弹出窗口的自定义快捷菜单。选择快捷菜单中的选项将把下面的 IFRAME 带到相关页面中。




单击下面段落中的加粗元素可以查看使用弹出窗口创建的词汇表。

使用 IE 5.5 创建动态的弹出元素现在变得前所未有的容易，Microsoft 的开发者已经将其实现于 Web 浏览器的最新版本中。要获得关于如何使用弹出对象和其它DHTML 特性的帮助，请看MSDN Web 工作室。
