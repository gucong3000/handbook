BorderRightColor Attribute

Internet Development Index

Right border color of an inline shape. Read/write VgColor.

Applies To

Shape

Tag Syntax

<v:element borderrightcolor="expression">

Remarks

Used by Microsoft Word.

Values include:

Value Description
Null No border color. Default.
color Color used for border.
This Indicates that Automatic was selected in the user interface of the application.

Microsoft Office Extensions Attribute
