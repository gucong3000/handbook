Command Identifiers

Internet Development Index

Command identifiers specify an action to take on the given object. Use them with the following methods:

  • execCommand
  • queryCommandEnabled
  • queryCommandIndeterm
  • queryCommandState
  • queryCommandSupported
  • queryCommandValue

Command Identifiers
